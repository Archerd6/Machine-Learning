Paso 3: PCA (Principal Component Analysis)

Otra forma de mejorar nuestro modelo es usando la reduccin de dimensin, podemos aproximar el conjunto de datos original con menos variables, mientras reducimos el poder computacional para ejecutar nuestro modelo. Usando PCA, podemos estudiar la razn de varianza explicada acumulativa de estas caractersticas para comprender qu caractersticas explican la mayor varianza en los datos.

Instalamos la funcin PCA y establecemos el nmero de componentes (caractersticas) que queremos considerar. Lo he configurado en "23" (total del nmero de caractersticas) para ver la variacin explicada de todos los componentes generados, antes de decidir dnde realizar el corte. Luego, "ajustamos" nuestros datos a la funcin PCA

Ajuste del nmero de componentes

from sklearn.metrics import confusion_matrix

def mostrar_resultados(y_test, pred_y):
    conf_matrix = confusion_matrix(y_test, pred_y)
    plt.figure(figsize=(3, 3))
    sns.heatmap(conf_matrix, xticklabels=["Malignant","Benign"], yticklabels=["Malignant","Benign"], annot=True, fmt="d", cmap="Blues", cbar=False);
    plt.title("Confusion matrix")
    plt.ylabel('True class')
    plt.xlabel('Predicted class')
    plt.show()
